# Workflows N8N

Esta pasta contém os arquivos JSON dos workflows utilizados no projeto TaskFlow.

## Como usar

1. Importe o arquivo JSON no painel do N8N (menu Workflows > Import).
2. Configure as credenciais e endpoints conforme necessário.
3. Consulte a documentação principal do projeto para detalhes de integração.

## Exemplos de arquivos
- taskflow-whatsapp.json: Workflow de integração WhatsApp + TaskFlow
- outro-workflow.json: Outro exemplo de automação

Adicione novos workflows nesta pasta e mantenha este README atualizado para facilitar o uso e manutenção.
